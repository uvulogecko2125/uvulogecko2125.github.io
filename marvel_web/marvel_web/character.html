美國隊長 美國隊長是創作於二戰時代的老牌英雄，最著名的反法西斯鬥士之一。他本是瘦弱的青年史蒂夫·羅傑斯，接受了超級士兵改造手術，成為唯一成功的實驗品，身體各項素質達到人類極限。他沒有真正的超能力，雖然使用著堅不可摧的特製盾牌，但他最大的武器當屬勇氣、毅力和愛國精神。二戰結束前夕他掉在冰海中休眠，幾十年後才被現代英雄找到。甦醒的羅傑斯繼續為了正義而戰，但當代世界已經不像昔日一樣善惡分明，他也不時被人們視為落後於時代的活化石作為Marvel少數近乎道德完人的英雄之一，美國上尉以自己與現實競爭的特點贏得了讀者的關注
鋼鐵人 開發軍火起家的百萬富翁花花公子托尼·斯達克，經歷生死考驗後決心把聰明才智用在造福人類上，為自己設計了一系列的高科技裝甲，成為當時最受矚目的英雄之一。斯塔克精明風趣、過著令書中人和讀者都羨慕的瀟灑生活，但他同時也面臨著身體和精神上的巨大壓力；雖然事業有成，遨遊政、經、軍界，卻也歷經多次重傷、破產、黑化、被控訴。他為世界和平和整個超級英雄事業做出過不可磨滅的貢獻，又因為自信過甚而試圖控制其他超人類，幾乎身敗名裂。他就是如此一個聰明、正直而又衝動、彷徨的商人，但其英雄本質仍然不會磨滅。
雷神 來自現實中的北歐神話，Marvel對其身份進行了一些現代性的設計，在保留神話色彩的同時把他連同全世界幾十個古代神話體系引入了漫畫時空。托爾是神界最強大，最有英雄氣概的成員之一，最初由於高傲而被貶到凡間，此後逐漸與人類為友，與各種惡勢力抗爭。他雖然是神，但不是全能的，也要面對形形色色的困境：與世俗規則的互不理解，與神族同胞的種種分歧，與宇宙更高勢力的抗爭……作為一個超凡而又漫步凡間的英雄，他將繼續在Marvel世界呼風喚雨
綠巨人 綠巨人浩克是60年代 Marvel最早創作的漫畫主角之一，是一位科學家布魯斯·班納受到自己開發的武器輻射而變異成的生物，具有非常巨大的力量，並以「越憤怒，越強壯」聞名。浩克代表了一種無視一切障礙、難以自控的野性，尤其是與循規蹈矩的文明人形成對比，因此他一直受到社會的敵視，與其他英雄的關係也時好時壞，但同時他仍然保持著鋤強扶弱的英雄特質。浩克長期以來受到廣泛的喜愛，人們把自己反抗權威、發泄不滿的感情寄托在他身上!
黑寡婦 黑寡婦（Black Widow）為Marvel漫畫中的一位超級英雄，本名娜塔莎羅曼諾夫，由斯坦李和唐氏兄弟共同創造的人物。從小是孤兒，由前蘇聯政府撫養，後被訓練成一名超級特工，反叛前蘇聯後加入復仇者。
鷹眼 他原是個在馬戲團長大的孤兒，在電視上受到鋼鐵俠啟發之下打扮成超級英雄，加入復仇者。擅使弓箭，百發百中，曾與黑寡婦是戀人的關係。在復聯1中開始被洛基控制神智，後被黑寡婦救下，與其他復仇者成員共同抗敵。
